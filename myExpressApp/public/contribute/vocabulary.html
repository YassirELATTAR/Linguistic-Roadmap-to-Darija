Contributing to the Vocabulary of Darija

Description of the Vocabulary Dataset

The collections of vocabulary consists of different themes. You can scroll down and check the category/ or from the menu. Since Darija does not have an official written form that generalizes it, the result of using it in texts (SMS, Social Media, Internet) was having various forms of the same word using Latin Alphabets. Hence, the use of MongoDB for storing this dataset is an ideal way since a word might have multiple spelling forms while another could have only one. The limit according to the available dataset is SIX different forms. They are labled as n1, n2, n3, n4, n5, n6. Besides the latin alphabets, there is an Arabic Form (using Arabic alphabts) and labeled as darija_ar, and finally a translation to English labeled eng.


For the latin forms (n1,..., n6), you don't have to include all of them: The minimum is 1 and maximum is 6.

Instructions:

  • Please make sure your data is in an XML File
  • Follow the structure below (select the category of the vocabulary you want to add) and watch the errors in case you get an alert!
  • Each <entry> is a record (lemma)

Example XML File:

          <corpus>

            <entry>

              <n1>SPELLING FORM 1</n1>

              <n2>SPELLING FORM 2</n2>

              <n3>SPELLING FORM 3</n3>

              <darija_ar>بالحروف العربية</darija_ar>

              <eng>ENGLISH TRANSLATION</eng>

            </entry>

          </corpus>
